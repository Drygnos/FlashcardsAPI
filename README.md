# Flashcards API

**Schéma de la base de données** :
https://drawsql.app/teams/team-rocket-39/diagrams/projet-flashcards

**Installation** :
```bash
npm install
```
**Configuration** :
Variables du .env : 
DB_FILE JWT_SECRET
**Lancement du projet** :
```bash
npm run dev
```
**Initialisation de la base** :
```bash
npm run db:push
```
**Seeding de la base** :
```bash
npm run db:seed
```
**Lancement de la base** :
```bash
npm run db:studio
```
